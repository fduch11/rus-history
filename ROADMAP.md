# История Руси — дорожная карта автономной разработки

## Цель

Создать автономное мобильное одностраничное веб-приложение «История Руси», которое работает локально при открытии `index.html`, публикуется через GitHub Pages, не использует npm, React, Vue, внешние библиотеки, CDN, API и `fetch`, хранит пользовательские данные в `localStorage` и остаётся работоспособным после каждого коммита.

## Правила автономной разработки

1. Работать напрямую в ветке `main`.
2. За одну итерацию выполнять один логически завершённый этап.
3. Перед коммитом проверять синтаксис `data.js` и `app.js`.
4. Проверять все локальные пути из HTML, CSS, manifest и service worker.
5. Не удалять работающие функции без равноценной замены.
6. После выполнения этапа обновлять этот файл и журнал итераций.
7. После итогового аудита отметить проект завершённым и больше не изменять репозиторий.
