# История Руси — дорожная карта автономной разработки

## Цель

Создать автономное мобильное одностраничное веб-приложение «История Руси», работающее при прямом открытии `index.html` и через GitHub Pages без npm, React, Vue, внешних библиотек, CDN, API и `fetch`.

## Правила автономной разработки

1. Перед изменениями читать `AGENTS.MD` и этот файл.
2. Работать напрямую в `main` и выполнять один логически завершённый этап за итерацию.
3. Перед коммитом проверять синтаксис всех JavaScript-файлов, локальные пути и запуск без сервера.
4. После этапа обновлять статусы, результат, следующий шаг и журнал.
5. После итогового аудита больше не изменять репозиторий.
